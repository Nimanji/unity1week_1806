# ギリ跳びユニティちゃん

## 概要

タイミングを合わせて向こう岸に跳ぶゲーム。

向こう岸の足場はけっこう小さいので、踏み切りの場所、ジャンプ力、角度をよーく狙ってユニティちゃんを飛ばせよう。

けっこう簡単な操作だけど、けっこうギリギリを狙わせるゲームにしていきたい。

## ざっくりとした操作

スペースキー ... 踏み切り位置の決定、各ゲージのストップボタン
